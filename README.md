# 🌦️ Weather App

## 📌 About
This is a simple Weather Application built using **HTML, CSS, and JavaScript**.  
The app allows users to search for any city and view real-time weather information using the **OpenWeatherMap API**.

It displays:
- Current temperature
- Weather condition
- Humidity
- Wind speed
- Weather icon

The project also includes input validation, error handling, and toast notifications for a better user experience.

---

## 🚀 Features
- Fetches real-time weather data
- Displays temperature in Celsius
- Shows humidity and wind speed
- Weather condition icons
- Toast notifications for errors and loading states
- Enter key support for quick search

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API
